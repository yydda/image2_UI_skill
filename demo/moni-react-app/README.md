# Moni React App Demo

This demo shows the expected Moni UI React delivery shape:

- Vite + React + TypeScript
- shadcn-style local components
- generated assets under `src/assets/generated`
- TypeScript asset imports in components

Run static validation:

```bat
.\validate.cmd
```

Run a full build after installing dependencies:

```powershell
npm install
npm run build
```
